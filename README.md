#### 介绍
你是否曾经幻想过：如果自己变成了一只或可爱，或帅气的兽兽，那该有~~多涩~~多好啊~
既然现实中做不到这些，那么这款模组或将会帮你满足这些愿望

##### 这个模组里有什么
这个模组目前包含了制作兽装的前置条件——DTD，与一张音乐唱片
未来一定会变得更加有趣，更加真实，更能满足各位的需求
